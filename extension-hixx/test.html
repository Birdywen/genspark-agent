HIXX AI Agent Bridge - 安装测试

如果扩展已正确安装，打开 hixx.ai 后右下角应该出现 Agent 面板。

安装步骤：

  1. 打开 Chrome，访问 chrome://extensions/
  2. 打开右上角「开发者模式」
  3. 点击「加载已解压的扩展程序」
  4. 选择目录：/Users/yay/workspace/genspark-agent/extension-hixx
  5. 确保扩展已启用（开关为蓝色）
  6. 刷新 hixx.ai 页面

检查扩展状态：

在 chrome://extensions/ 页面，找到 "HIXX AI Agent Bridge"：

  • 如果看到 错误 按钮，点击查看错误详情
  • 确认 ID 和版本号正确显示
